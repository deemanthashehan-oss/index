📐 Cone Geometry

🔴 Radius (r) — distance from center to base edge
🔵 Perpendicular Height (h) — vertical height from apex to base
🟢 Slant Height (L) — diagonal distance from apex to base edge
📐 Pythagoras Theorem
L² = r² + h²
h = √(L² - r²)    r = √(L² - h²)
